The data graphics research program seeks to further scientific and technical communication by improving the quality of data graphics.  

Current research topics include:

  • Developing and refining methods for data presentation

  • Improving methods for communicating design knowledge

  • Increasing graphicacy

Links to useful web pages and a basic set of references on graphical display, including the use of color can be found under Related Work.  Some color schemes that have desirable properties with respect to their interpretability by color-deficient viewers may be found under Example Color Schemes.
